Meet (React Application)

A Serverless Progressive Web Application

Meet App is a PWA (Progressive Web Application) built in REACT. The application gathers and displays upcoming developer-centric events in different cities worldwide by connecting to a Google Calendar API.

Designed to function across browsers, it also offers desktop installation and offline functionality providing users with a dynamic and accessible event exploration experience.

This project was written in TDD (Test-Driven Development), prioritizing test-writing before code implementation to ensure robustness and reliability.

CUB REST-Api

RESTful Api For A Movie Database

This project's objective was to build a RESTful API using Node.js and Express. Serving as the backbone, this API forms the base of both of my client-side movie database application, CUB Film Data (React) and CUB Film Data (Angular).

Integrating backend reliability with frontend innovation, this project showcases a symbiotic relationship between different technology stacks.

Developing the server-side of the application posed challenges, particularly in understanding backend processes and database management. I built this app from the ground up while learning the key concepts of backend programming. Being new to backend processes this task was challenging, and at the same time very enriching, resulting in a deeper understanding of backend programming and a newfound appreciation for database management.
