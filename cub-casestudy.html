Case Study

Working on CUB Film Data was an exciting encounter. The project was built in 2023 as a full-stack web application that displays information about arthouse movies, directors, and genres. Users are able to create an account, update their personal data, create a list of favorite movies, or delete their accounts.

Objective

The primary goal was to demonstrate mastery in full-stack app development within the MERN stack, showcasing proficiency in APIs, web server frameworks, databases, business logic, authentication, and data security. The project was aimed to be added to my professional portfolio.

Problem

The challenging task was to construct both the server-side and client-side components entirely from scratch. This involved addressing technical considerations, such as data management, user authentication, mastering backend programming concepts - as well as navigating through the server side with the terminal.

Approach

Server-Side:

Creating a robust REST API using Node.js and Express was the first step. I defined server-side endpoints for the API that support common HTTP methods (GET, POST, PUT, DELETE), providing users with access to movie, director, and genre data.

To connect the movie and user data, I connected a non-relational database (MongoDB) with Mongoose. The endpoints and HTTP methods were tested in Postman including the functionality of JWT authorization (see picture).

Finally, the API and the database were deployed to an online hosting service (Heroku) and a cloud-based database hosting platform (MongoDB).

Client-Side:

Transitioning to the client side, I used React to create an intuitive user interface.

Key components such as a main view (displaying all movies), a single movie view (details about a movie), and a profile view connect the data with the endpoints.

Implementing login and registration functionalities added depth to user interaction.

Challenges

The creation of the server side felt straightforward. While this was challenging and enriching at the same time, resulting in a deeper understanding of backend programming, the client-side implementation presented unique challenges and required a significant time investment compared to previous projects.

Learning React took me some effort, yet overcoming these obstacles with the help of my tutor Emanuel Okello enriched my skills in debugging and problem solving, and deepened my understanding of JavaScript and frontend development.
